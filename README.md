特朗普总统关于结束加沙冲突的全面计划


1. 加沙将成为一个去极端化、无恐怖的区域，不会对其邻国构成威胁。

2. 加沙将被重新建设，造福饱受苦难的加沙人民。

3. 如果双方同意此提案，战争将立即结束。以色列军队将撤回至商定的防线，为人质释放做准备。在此期间，所有军事行动，包括空袭和炮击，将暂停，战线将保持冻结状态，直到满足条件以实现分阶段的全面撤军。

4. 在以色列公开接受协议后的72小时内，所有人质（无论生还或遇难）将被归还。

5. 一旦所有人质获释，以色列将释放250名无期徒刑囚犯以及1700名在2023年10月7日之后被拘押的加沙人，其中包括在该背景下被拘押的所有妇女和儿童。对于每一名归还遗体的以色列人质，以色列将释放15具加沙人的遗体。

6. 一旦所有人质归还，承诺和平共处并上交武器的哈马斯成员将获得赦免。希望离开加沙的哈马斯成员将获得安全通道前往愿意接收的国家。

7. 一旦协议被接受，人道主义援助将立即进入加沙地带。援助数量至少应与2025年1月19日协议中规定的规模相当，包括重建基础设施（水、电、污水处理）、修复医院和面包房，以及引进清理废墟和打通道路所需的设备。

8. 援助物资和分发将在联合国及其机构、红新月会，以及其他与双方均无关联的国际机构监督下，不受干扰地进入加沙地带。拉法口岸的双向开放将遵循2025年1月19日协议中实施的同样机制。

9. 加沙将由一个临时的过渡性治理机构管理，即一个由技术型、无政治倾向的巴勒斯坦委员会组成的机构，负责为加沙人民提供日常公共服务和市政管理。该委员会将由合格的巴勒斯坦人和国际专家组成，并由一个新的国际过渡机构“和平理事会”监督和指导。该理事会由特朗普总统担任主席，其他成员及国家领导人（包括前英国首相托尼·布莱尔）将随后公布。该机构将制定框架并处理加沙重建资金，直至巴勒斯坦权力机构完成改革计划（包括2020年特朗普总统的和平计划和沙特-法国提案中所述内容），并能安全有效地重新掌控加沙。该机构将依照最佳国际标准建立现代高效的治理体系，以服务加沙人民并吸引投资。

10. 特朗普将制定一项加沙经济发展计划，通过召集曾帮助中东地区打造繁荣现代化城市的专家来完成。许多国际组织已提出有意义的发展和投资方案，将在安全和治理框架下进行整合，以吸引和推动投资，创造就业、机遇与希望。

11. 将建立一个特别经济区，与参与国协商优惠关税及准入政策。

12. 没有人会被强制离开加沙。愿意离开的人可以自由离开和返回。我们将鼓励人们留下来，并为他们提供建设更美好加沙的机会。

13. 哈马斯及其他派别不得以任何形式参与加沙治理。所有军事、恐怖及进攻性基础设施（包括地道和武器生产设施）将被摧毁且不得重建。加沙将接受非独立观察员监督的非军事化进程，包括通过协议规定的方式永久废弃武器，并通过国际资助的回购与再融入计划支持该进程。新加沙将全力致力于发展繁荣经济并与邻国和平共处。

14. 地区伙伴将提供保证，确保哈马斯及其他派别履行义务，并确保新加沙不会对其邻国或人民构成威胁。

15. 美国将与阿拉伯及国际伙伴合作，组建一支临时的国际稳定部队（ISF），立即部署到加沙。该部队将培训并支持经过审查的巴勒斯坦警察部队，并与约旦和埃及（在此领域有丰富经验）进行磋商。该部队将成为长期的内部安全解决方案。ISF将与以色列和埃及合作，帮助保护边境，并与新培训的巴勒斯坦警察部队一道，防止武器进入加沙，并促进物资快速安全流通以实现重建和振兴。各方将就冲突避免机制达成协议。

16. 以色列不会占领或吞并加沙。随着ISF建立控制和稳定，以色列国防军（IDF）将根据与非军事化相关的标准、里程碑和时间表逐步撤军，这些标准将由IDF、ISF、担保国及美国共同商定。目标是确保加沙不再对以色列、埃及或其公民构成威胁。在实际操作上，IDF将根据与过渡当局的协议，逐步将所占领的加沙地区移交给ISF，直至完全撤离，只在安全外围保留存在，直到加沙能够真正杜绝任何恐怖威胁的死灰复燃。

17. 若哈马斯延迟或拒绝此提案，上述计划（包括加大规模的援助行动）将在IDF移交给ISF的无恐怖区域推进实施。

18. 将建立一个基于宽容与和平共处价值观的跨宗教对话机制，努力改变巴勒斯坦人和以色列人的思维方式和叙事，强调和平带来的好处。

19. 随着加沙重建推进以及巴勒斯坦权力机构改革计划的切实执行，可能最终形成条件，开辟一条通向巴勒斯坦自决与建国的可信路径。我们承认这是巴勒斯坦人民的愿望。

20. 美国将推动以色列和巴勒斯坦之间展开对话，以达成和平繁荣共处的政治前景。
