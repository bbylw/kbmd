

**以色列公民们，**

我们今天庆祝的是我们在**复兴之战**中所取得的巨大成就之一。**让所有被绑架的人质——无论生死——全部回家。**

这是一个核心目标，一个我们自始至终坚持不懈的目标。在战争开始至今的两年时间里，我向人质家属承诺，也向你们，以色列公民们承诺：**我们将带回所有人，无一例外。**

我们承诺了，而且正在兑现。

我提醒你们，在战争初期，有人说：“好吧，我们需要适应一个现实，那就是我们无法带回人质，甚至一个活着的也带不回。”

**我的想法不同，我的做法也不同。**

我相信，如果对哈马斯施加**沉重的军事压力**，并结合**沉重的政治压力**，我们就能做到，我们就能带回所有的人质。

**而这正是我们所做的。**

但我也要告诉你们，毫不夸张地说，这并不容易。

我不得不面对来自内部和外部的巨大压力：压力是不要进入拉法，压力是不要控制**费城走廊**，压力是不要在其他战区采取行动。压力是要求停止战争并从加沙地带撤退，而哈马斯、真主党、阿萨德政权和伊朗都对此心知肚明。

但我坚决顶住了所有这些压力，因为我的眼前只有一个考量：**以色列的安全。**

这意味着实现战争目标，包括**释放人质**、**消除伊朗的核威胁和弹道导弹威胁**（它危及着我们在以色列的生存），以及**打破伊朗轴心**（哈马斯是其中关键的一部分）。

尽管有我提到的所有压力，我们仍在加沙继续战斗，在整个战区继续战斗。

在特朗普总统入主白宫之前，我们为以色列带回了 **158 名人质**，其中 **117 名是活着的**。我们部队通过勇敢的行动释放了部分人质，你们知道并了解他们，并向那些倒下的英雄们致敬。

在特朗普总统入主白宫之后，我们又带回了 **49 名活着和已故的人质**，其中 **10 名**是我们的部队通过额外的勇敢行动带回的。

但挑战依然存在，这是最艰难的挑战：**让所有剩下的人质回家。**

**20 名活着的，28 名已故的。**

你们还记得吗？一个又一个评论员，日复一日，都在告诉我们：“没有办法在不屈服于哈马斯核心要求的情况下带回人质，即以色列国防军必须完全撤出加沙地带，包括防御区、费城走廊，以及所有控制区。”

在这里，我的想法和做法也不同。

我知道，如果我们以强大的力量进入哈马斯剩下的最后一个据点——**加沙城**，并在那里摧毁其所有权力中心，哈马斯就会想要保住其统治。

我还认为，如果我们除了大规模军事压力之外，再增加一个**巨大的盟友**——**特朗普总统的巨大政治压力**，这种强大的组合将迫使哈马斯交出我们所有的人质。

只要**以色列国防军仍然深入加沙地带**，并**控制着所有要害之地**。

这就是正在发生的事情。

通过这种方式，我们正在**全面扼住哈马斯**，为接下来的阶段做准备。在接下来的阶段，哈马斯将被解除武装，然后被解散。

如果能通过简单的方式实现这一目标，那就太好了。如果不能，那就通过艰难的方式来实现。

我要用最清晰、最明确的方式表明：**任何说这份人质协议一直摆在桌面上的人，根本就没有说实话。**哈马斯从未同意在我们仍深入加沙地带时，释放我们所有的人质。

哈马斯同意这项协议，仅仅是在感到**刀架在脖子上**的时候。而且这把刀仍架在它的脖子上。

哈马斯同意这项协议，只是在**特朗普总统的计划**（我在华盛顿与他达成一致）以**前所未有的方式在国际上孤立了它**之后。

在当前计划的阶段，我们专注于**释放我们的人质**。

我必须告诉你们，在我过去两年与人质家属会面的那些**令人心碎的时刻**里，**我和我的妻子都流下了眼泪。**这无法不流泪。**心碎欲裂，我的心与他们同在。**

我向人质家属承诺：**我绝不放弃任何人，绝不放弃任何一个人。**

而且，**所有人质都将回到我们身边。**在未来的日子里，在上帝的帮助下，我们将带回所有人。

除了他们回归的巨大喜悦之外，他们还将面临身体和精神康复的艰巨任务。**整个以色列民族都将拥抱他们**，帮助他们重新站起来。

**我们将把已故的人质带回以色列安葬。**

我们将尽一切可能找到他们所有人的遗体，这是**互助互爱的神圣责任。**

**以色列公民们，**

在这个激动人心的日子里，我要感谢**以色列国防军和安全部队**的英勇战士们。自战争爆发以来，他们一直坚守在各个战线上，保卫着我们所有人的家园。

没有我们士兵们令人敬佩的牺牲，我们就无法将人质带回家。

我们**拥抱阵亡将士的家属**，带着深切的悲痛，对每一个家庭致以慰问。

我们祝愿我们亲爱的战争伤员——那些身体和心灵受伤的人们——**彻底康复**。

我要感谢**特朗普总统**的全球领导力，以及他为促成这项人质释放计划所作的**不懈努力**。

在这里，他再次向我们的人民和我们的国家展示了他伟大的友谊。

我感谢**特朗普总统的代表**，史蒂夫·温伯格和贾里德·库什纳，他们为达成这项人质释放协议付出了巨大的努力。

在我们这一边，向我们的首席谈判代表**罗恩·德尔默部长**以及整个谈判团队致以**最诚挚的感谢**。他们付出了**专注而富有创造性**的努力，才让我们走到了这一步。

**以色列公民们，**

两年前的住棚节，**妥拉节的喜悦**变成了**民族的悲痛。**

愿今年的妥拉节，在上帝的帮助下，变成**民族的喜悦**，**我们所有被绑架的兄弟姐妹们回家的喜悦。**

“回归家园”行动的名称取自《耶利米书》中对我们母亲拉结的应许：

“**你的儿女必回到自己的境界**。”

“**儿女们将返回他们的边境。**”

两年前，我们从**皮卡赫特防线**出发，向敌人发动了**城门之战**。

以色列公民们，感谢你们的**坚定不移和团结一致**，我们经受住了严峻的考验。我们取得了**巨大的胜利**，这些胜利正在**改变中东的面貌**。

但正如总参谋长所说，**这场战役尚未结束。**

我们面前仍然有**巨大的挑战**，但也有**巨大的机遇**。

我们将共同迎接挑战，共同抓住机遇。我们将共同完成我们的使命，**共同扩大我们周围的和平圈。**

谢谢你们，亲爱的公民们。
