 

根据多家媒体报道，娃哈哈集团董事长宗馥莉已于2025年9月12日正式辞去公司法人代表、董事及董事长等职务，并已通过集团股东会和董事会的相关程序获批。

知情人士透露，此次辞职的原因与商标使用“不合规”有关。

此外，宗馥莉的亲信严学峰（娃哈哈集团核心成员）于10月5日被解除立案审查，目前已返回宏胜集团正常上班。

此前，宏胜系公司已准备使用娃哈哈商标，但相关事宜仍在推进中。

娃哈哈官方已确认此事属实。

外界猜想：作为娃哈哈集团第一大股东，持股比例高达46%的杭州上城区文商旅投资控股集团有限公司（国有企业）过去多年来一直保持低调，几乎未深度介入公司运营。

鉴于近期高层变动频发、商标转让受阻以及股权结构的敏感性，未来该国有股东可能加强参与公司治理，例如通过董事会席位或专项监督机制介入，以维护国有资产安全、推动企业规范化发展，并化解潜在的家族传承风险。

这或将标志着娃哈哈从“家族企业”向更均衡的混合所有制模式转型。
