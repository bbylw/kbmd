# 时间的叛徒：民主党和极左如何利用昨日的眼泪，来勒索今天的美国

政治原则的保质期有多久？对于今天的民主党来说，答案似乎比牛奶的保质期还要短。他们正挥舞着政府关门的“核武器”，企图强行通过他们价值1.5万亿美元的激进左翼议程。而最具讽刺意味的是，这把武器正是由他们自己过去十年间的“谆谆告诫”和“声泪俱下”所铸造的。白宫网站上那份如同“罪证陈列室”般的时间线，让我们得以一窥这场登峰造极的政治虚伪。

让我们坐上时光机，用民主党自己的话语，来一场穿越时间的旅行：

**回到2013年**，当关闭政府的辩论上演时，参议员伊丽莎白·沃伦（2013年9月30日）将其定义为“那些无法在选举中获胜的人的最后手段”。参议员克里斯·墨菲（2013年9月27日）则站在道德高地上说：“辩论可以，但不能在联邦政府的资金和所有受其影响的生命悬而未决时。”那时的他们，是何等的“顾全大局”。

**快进到2018和2019年**，我们听到了更多“金玉良言”。伯尼·桑德斯（2018年1月4日）称这是“我们必须尽一切可能去阻止的严重而危险的行动”。艾米·克洛布切（2019年1月9日）言简意赅：“政府关门对经济不利。”柯尔斯顿·吉利布兰德（2019年1月8日）更是把调子拔高到人道主义层面：“这意味着美国儿童将饿着肚子上床睡觉。这是不可接受的。”

**再看看离我们更近的2023和2024年**，他们的警告变得愈发频繁和“真诚”。参议院领袖查克·舒默，这位今天的“关门总司令”，在短短一年前（2024年1月16日）还信誓旦旦地表示“任何理性的成员都不希望政府关门”。就在几天前，马克·华纳（2024年9月23日）还在警告此举会“削弱我们在全球舞台上的地位”。杰基·罗森（2023年9月29日）更是具体到“增加住房成本，剥夺老年人和儿童的食品援助”。

这些话语，跨越了十年之久，每一句都掷地有声，充满了对国家、对人民的“责任感”。然而，当时间来到**2025年的今天**，当他们手握权力，有机会通过同样手段来推进自己的议程时，所有的“原则”都瞬间蒸发了。

昨日的“严重而危险的行动”，变成了今天的政治筹码。昨日“不可接受”的儿童饥饿，变成了今天“可以承受的代价”。昨日对“无法赢得选举者”的鄙夷，变成了今天对自己行为的完美注解。这种转变是如此之快，如此之彻底，以至于人们不禁要问：你们当初的义正词严，究竟是发自肺腑，还是仅仅因为当时武器不在自己手中？

这正是这场政治闹剧的核心：民主党的原则是“情景式”的。当他们是少数派时，他们是稳定和秩序的捍卫者；当他们成为多数派时，他们就成了不惜一切代价的革命者。他们将自己塑造成受害者和保护者，只是为了在时机成熟时，能更心安理得地成为加害者。

共和党与特朗普总统提出的“干净的资金延期”方案，是一个避免混乱、维持基本运转的理性选择。但民主党拒绝了，因为他们的目的从来就不是治理，而是彻底的改造。他们不惜让军人领不到薪水，让退伍军人失去保障，让整个国家陷入停滞，只为换取一个推行其激进议程的机会。

历史是一面公正的镜子。它清晰地照出了这些“时间的叛徒”是如何背弃自己的言论和立场。他们昨日流下的“鳄鱼的眼泪”，如今正汇成淹没普通美国人生活的洪水。在这场精心策划的、跨越十年的虚伪表演中，民主党不仅是在勒索今天的美国，更是在嘲弄和侮辱他们自己的过去。
