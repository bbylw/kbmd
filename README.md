内塔尼亚胡独家访谈：阐述加沙和谈核心原则与以色列的“红线”
在即将与特朗普总统进行关键会晤之前，以色列总理本杰明·内塔尼亚胡接受了福克斯新闻的独家专访。他不仅证实了正在与美方团队就一份包含21点的和平计划进行合作，还明确界定了以色列在战后加沙治理、人质谈判以及地区安全上面临的不可退让的底线。

和平计划的协商与哈马斯赦免的条件
内塔尼亚胡指出，尽管和平计划尚未最终敲定，但以色列的核心目标坚定不移：释放所有人质、彻底解除哈马斯的统治和武装，并为加沙地带及整个地区开创新的未来。

针对报道中若达成协议，以色列将在48小时内释放1,000名巴勒斯坦囚犯以换取所有人质的条款，他表示细节仍在敲定。而对于哈马斯成员的赦免问题，他态度谨慎但开放：如果哈马斯领导人以释放所有人质并结束战争为条件被“护送出境”，这是一个可以讨论的选项。

然而，他强调以色列必须“完成任务”，彻底消除哈马斯的军事能力，防止其重演2025年10月7日的暴行。他进一步揭露，哈马斯正在通过抢劫人道主义食品并抬高价格，利用所得资金招募14、15岁的青少年作为补充兵员，以弥补其遭受的重大伤亡。

坚守“红线”：彻底拒绝巴勒斯坦权力机构（PA）回归
在谈到美国和平计划中关于让临时巴勒斯坦技术官僚小组管理加沙的提议时，内塔尼亚胡划下了不可逾越的“红线”，即拒绝巴勒斯坦权力机构（PA）在加沙的未来角色。

他重申了自己在联合国发言中的强硬立场，将10月7日后在以色列边境附近建立巴勒斯坦国比作“在9月11日之后，在距离纽约市一英里处给予‘基地组织’一个国家”，称其为**“纯粹的疯狂”（sheer madness）**。

总理指出，除非巴勒斯坦权力机构能彻底改变其性质——包括：

接受犹太国家。

停止“pay for slay”（为杀害犹太人的恐怖分子支付薪酬）政策。

教育其儿童拥抱与犹太国家的共存与友谊。

否则，他认为“改革后的”巴勒斯坦权力机构不具备可信度，不可能成为未来的治理伙伴。

卡塔尔空袭与亚伯拉罕协议的韧性
此次访谈还触及了以色列对卡塔尔境内哈马斯目标的空袭行动，该行动引发了美国和阿拉伯国家的“严重不满”。内塔尼亚胡坚称这是**“完全独立的以色列行动”**，目标是哈马斯，就像美国打击本·拉登一样。他拒绝在公开场合承诺未来不再进行此类袭击，强调此事需与特朗普总统进行外交讨论。

尽管阿联酋等国曾严厉谴责这些袭击是“严重违反国际法”和“不负责任的升级”，但内塔尼亚胡强调，由特朗普促成的亚伯拉罕协议（Abraham Accords）“具有惊人的韧性和强大”，在两年多的战争中仍保持良好。他相信，彻底击败哈马斯将进一步改善和扩大这些协议。

人道主义争议与核威胁应对
在人道主义问题上，内塔尼亚胡坚决否认以色列实施了“饥饿政策”。他指出，以色列已向加沙输送了数百万吨援助，相当于每人每天获得约1吨援助物资，可提供约3000卡路里的食物热量。他将所观察到的“匮乏”（deprivation）而非“饥饿”（starvation）归咎于哈马斯抢劫援助物资并袭击美方运营的分配点。他还引用了对数千名巴勒斯坦俘虏的检查结果，表示没有发现“任何一个瘦弱不堪的人”。

关于伊朗核威胁，他确认美国-以色列的联合行动旨在摧毁伊朗制造更多浓缩铀的能力和武器化核弹头的尝试，而非全部已有的浓缩铀库存。他高度赞扬了联合国近期因他的呼吁而恢复对伊朗的“快速恢复制裁”（snapback sanctions），认为这是对抗伊朗核野心方面的积极进展。

总结：人质与盟友关系
总理确认他相信仍有20名人质还活着，并强调以色列致力于带回所有活着的和已故的人质。

最后，内塔尼亚胡将他与特朗普总统的关系形容为“在我们两国历史上无与伦比的”，并赞扬了特朗普在其首个任期内承认耶路撒冷为以色列首都、将美国大使馆迁至耶路撒冷、承认戈兰高地主权以及退出灾难性的伊朗核协议等行动，称赞特朗普是一位“非常强大和独立的领导人”。
