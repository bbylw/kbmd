民主党主导的停摆恐致执法人员“生命安全面临威胁”且无薪水：顶级组织发警告
华盛顿 — 顶级执法组织警告称，美国政府一旦停摆，将使众多联邦执法人员的生命安全和生计面临危险。

执法组织发出紧急呼吁
全国警察组织协会 (NAPO)，一个代表美国 25 万州和地方执法人员的机构，周六在一份声明中发出警告：“如果国会议员们忽视他们的宪法职责，任由资金耗尽并导致政府停摆，那么那些致力于保护我们的城市和社区免受暴力犯罪、毒品和枪支侵害的联邦执法人员，将不得不冒着生命危险工作，却无法获得报酬。”

该组织进一步警告，联邦给各任务部队的资金将停止流动，同时，“联邦给州和地方执法部门的拨款、资源和支持也将停止，使重要的公共安全倡议陷入困境。”

NAPO 以本周早些时候达拉斯移民和海关执法局 (ICE) 设施发生的致命枪击事件为例，指出：“现在不是让联邦资金耗尽的时候。正如周三达拉斯 ICE 设施发生的致命枪击案所表明的那样，美国执法人员正在日益危险和充满挑战的环境中工作。”他们坚称，让这些人员冒着生命危险却得不到薪水，会损害他们工作的支持，并增加工作给他们及其家人带来的压力。

警察和监狱系统面临双重打击
代表全美 38.2 万成员的全国警察兄弟会 (NFOP) 也敦促国会履行其“最基本的责任”之一——“为联邦政府的运作提供资金”。

该组织主席帕特里克·约斯 (Patrick Yoes) 表示：“他们必须支付账单。”他指出，众议院本月早些时候通过的“干净的”持续决议案，需要在参议院获得 60 票才能通过，但最终以 44 票赞成、48 票反对的结果被否决。

约斯认为：“参议院的否决将给资助我们社区公共安全工作的项目带来重大干扰。司法部和国土安全部的政府官员将无法坚守岗位，为外勤行动提供支持。这对所有美国人来说都应该是不可接受的。”

地方监狱理事会也警告称，政府停摆“不仅会威胁”到联邦监狱局 (BOP) 服役人员的生计，还将“直接危及全国的机构安全和公共安全”。

该组织在声明中表示，BOP 在停摆期间不能减少工作时间、关闭大门或暂停运作。鉴于联邦惩教人员本就面临人员短缺，停摆将导致工作人员无薪工作，给他们的配偶和孩子带来巨大压力，并暗示这个职业“不稳定”，会使招聘更困难，并促使“有经验的警官到其他地方寻找工作”。

他们强调，在人手不足的监狱中，暴力事件会增加：“机构内部的失误会蔓延到公众——无论是通过走私、帮派协调，还是对执法人员和公民的威胁。”

政治僵局与行政部门的准备
这些呼吁是在白宫管理和预算办公室（OMB）向联邦机构发布备忘录之后发出的。该备忘录命令这些机构准备在政府停摆时进行“裁员”（reduction-in-force），特别是针对那些不属于唐纳德·特朗普总统优先事项的无资金项目，并起草终止计划。

地方监狱理事会坚称，这个问题“超越了党派之争”，因为“公共安全不能为了政治僵局而妥协。”

“停摆每持续一天，惩教人员就被要求冒着生命危险而得不到薪水，机构在日益增加的压力下运作，美国人民面临的风险就越大。不作为的代价远远超过任何可察觉的政治利益。”该理事会总结道。
