亲爱的乌克兰人民，乌克兰的男男女女们！今天，在保卫者日和圣母庇护节这一天，我们乌克兰向每一位参与保卫我们国家的人致敬，向每一位付出努力、让整个国家得以胜利的人致敬。我们赞扬数以百万计的人民，他们共同确保了乌克兰的生命延续。我们向国防和安全部队的每一个组成部分致敬，向我们的首批响应者致敬，向所有为我们国防而努力的人致敬，以及向所有以自己的方式保卫乌克兰、保卫自己的社区或身边的人致敬。

当然，今天也理应表彰我们在乌克兰的社区，那些在防御和坚韧中承担最大重担的城市。今天正是感谢这些城市的好日子。在 2022 年，我们设立了一个特别头衔，专为那些让乌克兰能够在那个极其艰难的年份中挺过来的城市。十座城市获得了“英雄城市”的称号：布查、沃洛诺瓦哈、戈斯托梅尔、伊尔平、马里乌波尔、米科拉伊夫、奥赫特尔卡、哈尔科夫、赫尔松和切尔尼戈夫。我们的每一座城市都象征着乌克兰最崇高的英雄事迹。

今天，在这个保卫者日，在圣母庇护节，我们还要表彰更多的英雄城市，英雄的城市。我签署了一项法令。这些城市拯救了我们成千上万人民的生命。这些城市在完全意义上保卫了我们的整个国家。

在第聂伯罗地区：巴甫洛格勒，一个属于乌克兰的力量之城。这座城市是一个具有战略重要性的疏散中心，疏散我们的民众；它也成为了前线伤员的医院基地。还有尼科波尔，一个真正的英雄工业城市，它在无情的炮击下仍然持续运转。马尔加涅茨——这座城市每天遭受打击，在俄罗斯炸毁卡科夫卡大坝后经历了巨大的困难，但它仍坚持了下来，这要归功于它的社区、整个第聂伯罗地区的英雄主义，以及每一个伸出援手的地区。

顿涅茨克地区的城市：德鲁日基夫卡，顿巴斯最光明的城市之一，现在被俄罗斯的炮击所摧残，但这座城市，这座城市，它在保卫自己，也在保卫他人。克拉马托尔斯克——我们工业的支柱之一，整个乌克兰东部最强大的中心之一。这座城市从 2014 年以来就一直在保卫国家。康斯坦丁尼夫卡——它抵抗着俄罗斯非凡的残暴，社区坚持不懈，城市本身也坚守不退。波克罗夫斯克——一位英雄。这位英雄对我们的防御意义重大，这座城市承载着我们许多英雄、许多战士，他们就在波克罗夫斯克地区、在波克罗夫斯克前线，在城市本身坚持着激烈的战斗，并以一种全球都能感受到的方式消灭着占领者——世界看到了这种英雄主义。斯拉维扬斯克——一个城市象征，是乌克兰防御开始的地方，这座城市今天为我们的战士注入力量，并致力于保卫顿巴斯。

扎波罗热地区的胡利亚伊波莱——拥有悠久的历史，它是我们乌克兰南方力量的世纪见证。今天，这座城市保卫着该地区和国家。奥里希夫——承受着持续的炮击，但我们的防线保持着非常坚固的阵地。

沃兹涅先斯克和巴什坦卡——米科拉伊夫地区的城市。这些城市的人民在面对一切时都没有屈服，他们进行了战斗，正是由于这些城市，乌克兰才得以挫败俄罗斯夺取我们南部的计划。

苏梅——英雄城市，英雄们的城市，是俄罗斯人准备占领甚至已经宣布要占领的城市，但他们没有得逞，也不会得逞。这是俄罗斯联邦最大的失败之一。

我们苏梅地区的特罗斯强涅茨——一个在 2022 年英勇坚守的城市，现在也投入了防御。正是在那里，在特罗斯强涅茨，在这场战争的第一年，俄罗斯向奥赫特尔卡、列别金和苏梅的广泛攻势被阻止了。

我们哈尔科夫地区的库皮扬斯克——一个现在情况特别困难的城市，战斗仍在继续，它遭受着俄罗斯的背信弃义和野蛮行径，但它被我们真正英勇的人民所保卫，这座城市英勇地为哈尔科夫地区抵御着俄罗斯。

最后是斯塔罗康斯坦丁尼夫，赫梅利尼茨基州的一个城市，它对我们的整个国家具有战略重要性，对我们的国防工业、对我们的航空业至关重要。在这场战争期间，斯塔罗康斯坦丁尼夫遭受了近 460 次俄罗斯袭击——但这座城市仍在英勇地坚持着。

每个人的坚韧，就是整个乌克兰的坚韧；民族的英雄主义，始于我们每个人内心的勇气。我为你们骄傲！很荣幸能代表这样的民族，这样的城市，这样的社区，这样勇敢的人民。

荣耀归于乌克兰！
