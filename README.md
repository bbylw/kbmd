鲍里斯·约翰逊：特朗普以纯粹的力量在加沙促成和平。现在他需要对普京发出严厉警告——我相信他会做到
仍有许多可能出错的地方。剩余的人质尚未返回。以色列人只是刚刚开始从加沙的部分地区撤军。

目前仍不清楚哈马斯是否已经明白他们真的必须解除武装。也不清楚内塔尼亚胡政府是否已经明白，关于巴勒斯坦人民运行自己生活的正当愿望的讨论也必须开始。未来几天仍将充满挑战。

但这些困难是不可避免的，它们不应该让我们忽视已经取得的巨大成就。

在过去的两年里，以色列和哈马斯之间的战争就像世界颅骨中一颗困扰世界已久的剧痛，有时消退，但又会再次爆发，让我们所有人都近乎抓狂。

我们感受到了对 10 月 7 日大屠杀的愤怒和悲痛，对哈马斯暴行的厌恶与憎恨，对巴勒斯坦人似乎永无止境的苦难感到恐惧，以及一直以来对人质能够回家的强烈渴望。

我们不应低估这一时刻的喜悦，它给以色列家庭和加沙带来的慰藉。我们终于看到了前进的道路。

我们正走在和平的道路上。这是一种美好的感觉，不管你喜不喜欢，这在很大程度上要归功于唐纳德·特朗普。

正是这位美国总统利用他的职权和个人魅力促成了这件事。他是通过对双方施加最大压力来做到这一点的。他向持怀疑态度的比比·内塔尼亚胡施加了压力，内塔尼亚胡对哈马斯的意图持有合理的怀疑，现在他将努力维持他的政府。

他的一些部长不仅反对这项协议，还希望以色列吞并加沙。他们憎恶巴勒斯坦国的想法。有人会认真相信，如果任由内塔尼亚胡自己决定，他会与哈马斯达成这项协议吗？不，当然不会。这项协议需要特朗普。

与他的一些前任不同，特朗普在以色列建立了巨大的信誉和信任。他曾将美国大使馆迁至耶路撒冷。他曾通过支持以色列成功轰炸伊朗核浓缩设施而震惊世界。

在特朗普和内塔尼亚胡的每一次对话中，都潜藏着这样一个现实：以色列的防御最终依赖于美国的资金。由于双方都知道这种支持不是无条件的，特朗普处于达成协议的独特位置。

时机成熟时，他对内塔尼亚胡发出了严厉警告，而内塔尼亚胡之所以能够同意，是因为哈马斯毫无疑问地做出了让步。

有人会认真相信，如果任由哈马斯自己决定，他们会做成这笔交易吗？他们必须放弃对以色列唯一的筹码：人质。他们将无权参与加沙的运作。

这项协议对哈马斯来说是正确的，但对许多人来说，这感觉像是失败。

他们部分是被激励措施说服的——停止轰炸、对其罪行进行大赦和获得投资。但哈马斯最终来到谈判桌的原因是来自特朗普可信的威胁——如果他们错失了这个机会，他们将彻底被消灭。他们看到了真主党和伊朗的遭遇，他们得出结论，特朗普言出必行；他将支持以色列到底，而他们将被碾成齑粉。

**特朗普以实力促成了和平。**他无情地施加了美国压力。现在，全世界都需要他再做一次。

有一场战争，在过去三年半的时间里，其造成的死亡和痛苦让加沙黯然失色，那就是乌克兰战争。

特朗普可以通过果断地施加美国压力——向普京施压，并向美国的盟友施压，要求他们与特朗普的行动保持一致——来结束这场战争，就像他为加沙带来和平一样。

我们看到了他是如何向乌克兰人施压的。它奏效了。他达成了一项出色的矿产交易，而且乌克兰总统弗拉基米尔·泽连斯基和白宫之间的关系比以往任何时候都好。

我们看到了他是如何向欧洲人施压的。它奏效了。事实上，所有其他北约国家都在大幅增加国防开支。

到目前为止，我们还没有看到对俄罗斯施加类似的压力。特朗普在阿拉斯加为普京铺设了红地毯，向他表示了所有可能的礼貌。但他除了蔑视之外，没有得到任何回报。

普京对美国、北约和西方竖起了两根手指。他明确表示，他将继续杀害无辜的乌克兰人，直到他相信北约——尤其是白宫——认真对待阻止这场战争，并认真对待一个自由、主权和独立的乌克兰。

我知道特朗普关心这两件事：阻止普京的袭击和维护一个自由的乌克兰。他知道，如果他允许西方在乌克兰战败，他就不能指望效仿罗纳德·里根——这位冷战的胜利者。现在是我们所有人——英国、欧洲、北约——帮助他对普京施加压力的时候了。

让我们立即解冻那 2500 亿美元的俄罗斯资产，并将其作为普京赔款的预付款交给乌克兰。这笔现金仍然主要存放在布鲁塞尔的一个银行账户里，这是可耻的。

让我们给乌克兰人他们需要的装备，以摧毁普京发起邪恶袭击的基地，不仅是战斧导弹，还有仍未送出的 1000 枚德国“金牛座”导弹；让我们允许乌克兰人使用他们已有的远程武器。

让我们打击普京规避制裁的油轮船队，如果其中一艘沉入海底，那也不是一件坏事。

让我们支持特朗普，对那些仍在购买俄罗斯石油和天然气并助长普京屠杀的国家进行经济惩罚。

至于“有意愿的联盟”，即在乌克兰的西方地面部队，让我们现在就部署吧。在任何情况下，这些部队都不会被部署用于作战。它们部署在远离前线的地方，提供后勤和训练支持。

由于乌克兰大部分地区是安全的，它们现在就可以前往，哪怕只是为了向莫斯科表明一个关键点：由乌克兰人来决定哪些外国军队可以访问他们的国家——而不是由普京决定。

这场战争的根本在于乌克兰的命运和自由；我们现在就可以向普京表明，乌克兰必须永远可以自由选择自己的命运。这就是为什么我们应该立即部署，而不是等待普京同意，否则“有意愿的联盟”将变成“等待的联盟”。

我们之所以有望实现加沙和平，是因为特朗普最终施加了关键压力。双方都意识到美国的耐心已到尽头，妥协是最好的选择。

双方都意识到，如果他们未能达成协议，他们将面临更糟糕的结果。

乌克兰的杀戮已经持续了足够长的时间。现在是普京感受到特朗普可以施加的压力的时候了。

毋庸置疑，当他这样做时——我相信他会的——那个诺贝尔奖将是板上钉钉的事。
