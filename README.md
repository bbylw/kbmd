马克·莱文评论：坎摩尔事件与言论自由的真正威胁
引言：暴力与煽动宣传的对决
在这段时长超过二十分钟的激烈评论中，主持人马克·莱文（Mark Levin）首先将焦点聚焦于政治暴力与言论自由两大主题。莱文直言不讳地将民主党描述为一个**“暴力政党”，并指出**其历史根基可追溯到美国的黑暗时期：奴隶制、种族隔离、三K党、优生学和吉姆·克劳法。他强调，1960年代的骚乱、新左派及其学生运动的暴力倾向，均源自民主党的意识形态。

随后，莱文引用了他自己著作《论权力》（On Power）中的观点，深入阐述了负面权力如何通过**“煽动宣传”（agitprop）来控制人群。这种策略，莱文将其定义为一种结合了操纵、欺骗、重复、隐瞒、转移注意力和散布恐惧的沟通方式。其目的不仅在于掌握对语言的权力，更在于在毫无道德约束的前提下控制民众**，旨在唤醒偏见、扼杀独立思想，并通过制造对非信徒的“暴力暗示”和“含蓄威胁”，来颠覆公民社会，激起群体间的战争，并最终服务于煽动者的政治目的。现代的马克思主义者、民主社会主义者以及其他极端分子使用的正是这种极权主义工具。

对吉米·坎摩尔停职事件的抨击
莱文将脱口秀主持人吉米·坎摩尔（Jimmy Kimmel）的停职和复职事件，定性为一场与真正的言论自由毫无关联的政治闹剧。

他坚称，坎摩尔最初被撤职，并非因为政府控制言论，而是由于他本人**“卑鄙可憎”——他发表了关于政治暗杀查理·柯克**（Charlie Kirk）的“极不恰当的评论”，此举是“令人作呕”的。莱文指出，柯克是因为行使言论自由而被谋杀，而坎摩尔却将矛头指向了柯克的支持者。

莱文强调，这是一起私营企业与员工之间的纠纷，与《第一修正案》无关，后者仅限制政府行为。ABC和迪士尼作为私营平台，完全有权根据行为准则约束员工。莱文进一步指出，FCC主席在其中并未扮演任何角色，否认了任何政府干预的可能性。坎摩尔的停职是由于**“我们人民”**的即时反感和附属电视台的撤播所致。

至于坎摩尔的复职，莱文断言这并非是出于广泛的观众压力，更没有财务上的必要性。坎摩尔被带回，正是因为他是**“自由派宣传家”，受到民主党活动家、好莱坞和民主党媒体的喜爱，是这些群体的政治压力导致了ABC的决定。莱文最终结论：坎摩尔不是言论自由的受害者，而只是他“自己的大嘴巴、仇恨和自恋”**所造成的后果的受害者。

谁是言论自由的真正威胁？
莱文随后将矛头转向那些在坎摩尔事件中支持其自由的民主党人和左派，指责他们实际上憎恨并长期攻击《宪法》和言论自由。莱文断言，真正的言论自由危机正是由左派和政府干预造成的。他列举了以下历史和当代事例来佐证：

当代政府干预（拜登政府的行动）：
针对保守派媒体的攻击： 他回顾了针对拉什·林博（Rush Limbaugh）和福克斯新闻的持续攻击，并指出奥巴马政府曾有一个计划试图取消保守派谈话电台。在民主党国会议员致信有线电视平台要求取消福克斯新闻时，主流媒体和喜剧演员们并未伸出援手。

FACE Act与反堕胎活动家： 司法部（DOJ）曾利用《FACE Act》起诉非暴力的反堕胎活动家，甚至将一些在堕胎诊所前抗议的老年女性投入监狱。最终，唐纳德·特朗普赦免了其中23人。

针对宗教团体的审查： 莱文披露，联邦调查局（FBI）针对“激进传统天主教徒”的审查行动规模远大于官方承认的范围。参议员查克·格拉斯利（Chuck Grassley）公布的文件推翻了前FBI局长克里斯托弗·雷（Christopher Wray）关于该备忘录是“孤立事件”的说法。

杭特·拜登“笔记本”事件的审查： 莱文揭露了，FBI花费了近一年的时间来准备社交媒体平台，以审查杭特·拜登（Hunter Biden）笔记本电脑的故事，同时向这些公司隐瞒了有关笔记本电脑真实性的信息，这严重违反了《第一修正案》的精神。

行政命令与家长： 他提到了梅里克·加兰（Merrick Garland）的备忘录，该备忘录是联邦政府行政部门协调一致的行动，旨在针对学校董事会会议上抗议的美国父母，并威胁要监禁他们。

最高法院裁决： 莱文批评了最高法院在“墨菲诉威斯康星州”（Murphy v. Wisconsin）一案中推翻下级法院禁令的决定。大法官托马斯（Thomas）、阿利托（Alito）和戈萨奇（Gorsuch）在异议中将此案描述为联邦官员针对对COVID-19持异见者进行的**“影响深远且广泛的审查运动”**。

特朗普的“禁言令”： 莱文指出，法院对总统候选人唐纳德·特朗普实施“禁言令”是荒谬的，这使得一位在总统竞选中的被告无法对美国人民发表言论。

历史上的民主党总统对言论自由的侵犯：
伍德罗·威尔逊： 实施了《1918年煽动叛乱法》（Sedition Act），使其成为非法行为，并起诉了数千名反战活动家；他还授权邮政总局审查“不爱国”的出版物，并成立了政府宣传机构。

富兰克林·罗斯福： 莱文明确指出，罗斯福曾利用国税局（IRS）调查并针对他的媒体批评者，并通过国会委员会获取了西联汇款（Western Union）数万份私人电报。

结论
事实是，唐纳德·特朗普并非言论自由的威胁。他是一个对那些一直憎恨言论自由、并散布恶毒仇恨言论的既得利益者及其意识形态的威胁，而正是这些言论和政府行为正在对美国社会造成可怕的后果。
