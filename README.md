2025年诺贝尔生理学或医学奖揭晓：美日三位科学家因发现免疫“刹车”机制获奖
2025年10月6日

卡罗林斯卡学院诺贝尔大会决定将2025年诺贝尔生理学或医学奖授予：

玛丽·E·布伦科 (Mary E. Brunkow)
美国西雅图，系统生物学研究所

弗雷德·拉姆斯德尔 (Fred Ramsdell)
美国旧金山，索诺玛生物疗法公司

坂口志文 (Shimon Sakaguchi)
日本大阪，大阪大学

“以表彰他们在有关外周免疫耐受方面的发现”

他们发现了免疫系统如何被有效控制

人体强大的免疫系统必须受到调控，否则它可能会攻击我们自身的器官。玛丽·E·布伦科、弗雷德·拉姆斯德尔和坂口志文因其在外周免疫耐受方面的突破性发现而被授予2025年诺贝尔生理学或医学奖，这些发现揭示了防止免疫系统伤害身体的机制。

每天，我们的免疫系统保护我们免受成千上万种试图入侵身体的微生物的侵害。这些微生物形态各异，许多还演化出与人体细胞的相似性作为一种伪装。那么，免疫系统是如何判断应该攻击什么、又应该保护什么呢？

玛丽·布伦科、弗雷德·拉姆斯德尔和坂口志文因其与外周免疫耐受相关的基础性发现而荣获2025年诺贝尔生理学或医学奖。这几位获奖者识别出了免疫系统的“安全卫士”——调节性T细胞，它们能够阻止免疫细胞攻击我们自己的身体。

诺贝尔委员会主席奥勒·坎佩（Olle Kämpe）表示：“他们的发现对于我们理解免疫系统如何运作，以及为何我们并非人人都会患上严重的自身免疫性疾病，起到了决定性的作用。”

1995年，坂口志文逆流而上，做出了第一个关键发现。当时，许多研究人员坚信，免疫耐受的形成仅仅是因为潜在有害的免疫细胞在胸腺中通过一个称为“中枢耐受”的过程被清除了。坂口志文的研究表明，免疫系统更为复杂，他发现了一类前所未知的免疫细胞，这类细胞能够保护身体免受自身免疫性疾病的侵害。

玛丽·布伦科和弗雷德·拉姆斯德尔在2001年做出了另一项关键发现，他们为一种特定的小鼠品系为何特别容易患上自身免疫性疾病提供了解释。他们发现这些小鼠在一个被他们命名为 Foxp3 的基因上存在突变。他们还证明，该基因在人类中的对应基因发生突变，会导致一种严重的自身免疫性疾病——IPEX综合征。

在此两年后，坂口志文成功地将这些发现联系起来。他证明了 Foxp3 基因调控着他在1995年所发现的那类细胞的发育。这些细胞现在被称为调节性T细胞，它们监控着其他免疫细胞，确保我们的免疫系统能够耐受自身的组织。

获奖者们的发现开创了外周耐受这一研究领域，并推动了针对癌症和自身免疫性疾病的医学疗法的发展。这也可能使得器官移植更为成功。其中一些疗法目前正在进行临床试验。
