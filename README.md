新西兰外长联合国发表演讲 - 新西兰不承认巴勒斯坦国
  

12个月前，我们在发表新西兰国家声明时曾表示，应对我们所面临的国际挑战，需要多少政治领导力，现在比以往任何时候都更加明显。我们需要政治领导力来恢复国内和国际机构的信任，以建立团结、维持脆弱的社会凝聚力，并填补国际社会通过联合国无法解决的空白。

回顾过去一年发生的事件，这种领导力差距有增无减。在过去80年里维持和平的国际秩序正处于崩溃的边缘。国际秩序从规则转向权力的转变正在继续其恶性发展，而安全理事会在许多紧迫的地缘政治挑战上实际上陷入了瘫痪。这种无法采取行动的局面，很大程度上是由于常任理事国五国的否决权所致，它深刻影响了人们对联合国更广泛合法性的认知。

对于像新西兰这样的小国来说，我们过去80年的安全和繁荣一直依赖于一个运转良好的多边体系，这一体系的侵蚀令人深感不安和代价高昂。联合国作为一个组织，自八十年前成立以来规模已大大增加，但其有效性并未跟上这种增长。因此，它成为了那些质疑其雄心广度的人，以及那些想要削弱多边体系本身的人的更大目标。

联合国的改革努力需要适应其当前的结构性、财政和地缘政治挑战的规模。我们将作为授权执行审查的联合主席发挥我们的作用，因为新西兰作为一个小国，希望联合国进行大胆的改革，以恢复其在成员国中的地位。

新西兰对我们全球范围内看到的人道主义灾难深感不安。尽管国际社会的焦点理所当然地集中在加沙和乌克兰的苦难上，但刚果民主共和国、叙利亚、苏丹和缅甸等国家的人道主义危机令我们更加深切担忧。我们担心的是，大规模暴力、流离失所和饥荒的影响，将在那些已被政治不稳定和冲突摧毁的国家中，制造进一步的代际暴力循环。

在叙利亚，近1700万人需要人道主义援助，有同样数量的人流离失所。在苏丹，超过2100万刚果人需要人道主义支持，超过3000万人（约占其人口的65%）急需人道主义援助和保护。基本服务已经崩溃，疫苗接种率暴跌，包括性暴力在内的暴力行为普遍存在。约3000万苏丹人流离失所。在缅甸，超过350万人国内流离失所，2200万人需要人道主义援助，使其成为东南亚最严重的人道主义危机。

这些惊人的数字令人震惊，因为它们表明在太多地方，我们共同的人性受到蔑视，甚至更糟。我们需要能够创造可能性而非扼杀可能性的领导力；需要说服而非控制的领导力；需要诉诸我们人性中更美好一面而非最坏本能的领导力。最重要的是，我们需要满足对住所、食物、和平生活、感到安全以及希望孩子能茁壮成长的共同愿望的领导力。

最重要的是，我们需要勇敢的领导力。正是勇敢的领导力，让埃及总统安瓦尔·萨达特于1977年11月在耶路撒冷的以色列议会发表演讲，赞颂和平。萨达特将为此付出最终的牺牲，但他的国家和以色列在随后的几十年里都受益于他的勇气。

在那次演讲中，萨达特说，我引用：“这场将我们从战争带向战争，从受害者带向更多受害者，直到您和我今天都达到了可怕深渊和恐怖灾难的边缘的斗争，除非我们共同抓住今天这个基于正义的持久和平的机会。”引用结束。

我们再次处于加沙可怕深渊的边缘，但我们不禁要问，能够将以色列人和巴勒斯坦人从深渊中拉回、为这场令人难以忍受的暴力的受害者投射希望的领导力在哪里？因为没有希望，助长和延续这场无休止冲突的暴力循环就不会结束。它只会成为一个恐怖分子滋生地，为那些只有仇恨的人招募下一代新兵。

这就是新西兰政府正在考虑巴勒斯坦建国这一棘手问题的当前背景。新西兰是两国方案和巴勒斯坦自决权的坚定支持者。我们长期以来的政策是：承认巴勒斯坦国是何时发生的问题，而非是否发生的问题。

我们注意到，一些国家在七月和八月宣布打算承认巴勒斯坦建国，本周又再次确认。我们还密切观察了以色列政府在这些早期宣布之后在加沙和西岸的行动，并认真听取了本周早些时候在两国方案会议上提出的论点。

然而，承认巴勒斯坦建国之所以异常复杂，是因为它深深地嵌入在一个看似难以解决、永无休止的冲突局势中。很少有新西兰人能在他们的一生中回忆起中东有任何一段可持续的和平时期。虽然新西兰离中东的冲突最远，但我们也承认人们对此抱有的强烈看法。

新西兰人对哈马斯于2023年10月7日对以色列公民进行的暴行感到震惊，这是以色列历史上最严重的屠杀。哈马斯在未来的巴勒斯坦国中没有计划，也没有立足之地。他们只懂得仇恨。

今天，在恐怖事件发生近两年后，包括哈马斯继续扣押以色列人质，我们对加沙令人心痛的饥荒景象感到震惊。我们同样对以色列政府严重不成比例的反应感到厌恶。

然而，有一句关于乐器的老话，很好地概括了巴勒斯坦建国承认的棘手问题：如果琴弦拉得太紧，它会断裂；但如果太松，乐器就无法演奏。

那些希望他们早前发出的承认巴勒斯坦国信号能够保护和促进两国方案的国家，反而看到了以色列政府的“断裂”，继续其在加沙广受谴责的军事行动，同时继续在西岸开发违反国际法的非法定居点。

新西兰政府同意合作伙伴所寻求的目标，并承认他们的良好意图。我们赞扬那些试图通过外交努力结束暴力的国家的领导力。我们迫切希望外交取得成功，我们相信拥有影响力的国家最有可能取得突破，展现全球领导力。

然而，我们不认为当前局势是维护两国方案的最后或甚至最好的机会。相反，我们认为未来的情况，即以色列和巴勒斯坦的政治领导力成为资产而非负债，并且其他情境变量已将当前的算计从冲突转向和平，将更有利于承认巴勒斯坦建国。

这正是我们现在决定是否承认巴勒斯坦建国的两难境地，因为建国承认作为一种和平工具，现在也无法发挥作用，因为没有一个完全合法和可行的巴勒斯坦国可以承认。巴勒斯坦并未完全满足公认的建国标准，因为它不能完全控制自己的领土或人口。

国际社会中更多国家承认巴勒斯坦国与保护两国方案这一既定目标之间，也没有明显的联系。事实上，自从合作伙伴发表声明以来，我们观察到的情况表明，现在承认巴勒斯坦很可能适得其反：即哈马斯抵制谈判，认为自己正在赢得全球宣传战，同时将以色列推向更加顽固的军事立场。

我们还认为，在这个时候进行承认，也可能被哈马斯和以色列双方进行政治操纵。哈马斯会试图将我们对巴勒斯坦的承认描绘成一次胜利，正如他们对合作伙伴的宣布所做的那样。以色列则会声称承认奖励了哈马斯，并解除了他们释放人质和同意停火的压力。

女士们，先生们，因为即使在这次暴行发生近两年后，哈马斯仍拒绝交还人质。

因此，就像此前150多个国家一样，新西兰现在承认巴勒斯坦建国，只不过是针对不可改变的现状的一种微不足道的存在主义的反抗姿态。我们尚未准备好做出这种姿态。

相反，新西兰政府认为，它只有一次机会承认巴勒斯坦建国，最好在能够提供更大和平与谈判前景的条件出现时这样做，而不是像现在这样战火肆虐、哈马斯仍在位且下一步行动不明确的情况下。我们不认为现在是时候。

自10月7日袭击以来，我们一直反复要求停火、释放剩余人质，并要求以色列允许重要援助流入加沙。我们的焦点仍然集中在这里。

为此，今天新西兰政府宣布将进一步提供重大的财政捐助以支持加沙的人道主义危机。我们的焦点不会从加沙现在最需要的地方转移。援助必须流动，暴力必须停止。

然而，归根结底，如果领导人及其人民不珍视和平，或者无法克服他们的仇恨，我们采取的任何行动都无法将他们从破坏性和毁灭性的道路上转移开。但是，如果他们珍视和平，当我们看到不是口头承诺而是协议形式的行动，以及当建国及其支持机构出现时，新西兰届时将承认巴勒斯坦国。

各位同仁，我们对这个动荡世界中存在的领导力差距的关注，直接关系到我们作为成员国（无论大小）的责任，即支持联合国，确保正如《联合国宪章》序言所宣告的那样，使后世免遭战祸。

自联合国在旧金山创建以来，我们对它的支持始终坚定不移。保持旧金山精神的活力是我们所有人的工作，因为正如有人曾说，联合国仍然是我们对未来和平的最大希望。现在是我们这些相信多边主义的人，不是去妥协它，而是站起来为它而战的时候了。谢谢。
