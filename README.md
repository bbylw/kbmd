 

2025年诺贝尔化学奖得主日本化学家 北川进：挑战中的化学梦想家——一位京都学者的传奇人生
2025年10月8日，瑞典皇家科学院宣布，京都大学名誉教授北川进与澳洲的理查德·罗布森及美国的奥马尔·亚吉共同获得诺贝尔化学奖，以表彰他们“开发金属有机框架”（Metal-Organic Frameworks, MOFs）的贡献。这项奖项不仅肯定了北川进在多孔材料领域的开创性工作，更点亮了无数年轻研究者的梦想。在当天的诺贝尔新闻发布会，74岁的北川进分享道：“我告诉我的学生，挑战在化学中、在科学中非常重要。”这句话不仅是他的对后辈的勉励，也折射出他一生追求“有用之无用”的哲学人生轨迹。

京都出生的“化学侦探”
1951年7月4日，北川进出生于日本京都，这座古都不仅是他的家乡，也成为他一生的精神归宿。作为京都人的他，从小浸润在传统文化与现代科学的交汇中。京都大学的iCeMS（国际细胞材料科学研究所）官网上，他自述：“我来自京都城。”这种根植于本土的归属感，伴随着他走过学术生涯的每一步。

北川进选择化学作为职业的理由颇具趣味。他曾笑言：“我选择化学作为事业，因为化学家能分辨乙醇和甲醇的差别。”这句看似轻松的回答，背后隐藏着他对科学精准性和好奇心的热爱。1970年代的京都大学本科阶段，北川进接受了广博的教育课程，包括历史、欧洲语言、基础科学以及哲学导论。其中，最深刻的影响来自中国古代哲学家庄子的著作。庄子在《齐物论》中论述“无用之用”，强调看似无用的空间或事物往往蕴藏巨大价值。这一思想后来成为北川进研究MOFs的灵感源泉——那些分子间的“空隙”，看似无用，却能储存气体、过滤污染物，改变世界。

本科期间，北川进有幸进入著名化学家米泽贞次郎的实验室工作。米泽曾师从1981年首位亚洲诺贝尔化学奖得主福井谦一，这段经历让年轻的北川进近距离接触到顶尖科学的脉动。1979年，他以烃化学为题获得京都大学博士学位，正式踏入学术界。

从边缘到前沿：马基雅维利的“直觉金矿”
博士毕业后，北川进的职业生涯起步于大阪近畿大学，1979年至1983年担任助理教授，随后升为讲师（1983-1988年）和副教授（1988年）。1992年，他转任东京都市大学无机化学教授。这些年份看似平稳，却是他酝酿突破的关键期。1986年至1987年，他作为客座教授赴美国德州农工大学与著名化学家F. Albert Cotton共事，这是他人生的首次国际历练，也开阔了他的视野。除了这一年，他几乎将整个职业生涯献给了日本本土，这份坚持让他深植于京都的文化土壤中。

1998年，北川进重返母校京都大学，任无机功能化学教授，从此开启了研究生涯的黄金时代。他的研究聚焦配位化学，尤其是有机-无机混合化合物的化学与物理性质。1997年，他发现配位聚合物具有气体吸附特性，这一“直觉”被后世视为MOFs领域的里程碑。回顾这段经历，北川进在iCeMS的“先锋访谈”中自嘲道：“当我首次发表这项工作时，人们视我为马基雅维利（maverick，异类）。”他凭借一股“预感”，转向一个当时鲜有人问津的方向，结果“掘到金矿”——MOFs不仅能吸附小分子气体，还能实现大规模结构转变，应用于碳捕获、药物传输等领域。

这条道路并非一帆风顺。作为“异类”，北川进曾面临质疑，但他坚持“挑战”的信念。2007年，他共同创办iCeMS，并于2013年出任所长至今。即使在2025年诺贝尔发布会，他仍谦逊地说：“我深感荣幸。我的梦想是捕捉空气并分离它，用于环境保护。”这不仅是科学梦想，更是对手未来人类的承诺。

实验室的诗意：侦探小说与歌舞伎的化学家
北川进的个人生活同样充满诗意。他最喜爱的实验是金属络合物的结晶化，“这是我们化学的起点”。最爱的理论是边界轨道理论（Frontier orbital theory），因为它为合成化学家提供指南。最偏好的仪器是固态NMR，“无需任何处理就能观测光谱”。

闲暇时，他化身“化学侦探”：热爱侦探与警匪小说、“在虚构世界中成为侦探”；迷恋歌舞伎与超歌舞伎，“人类本性的表现令人惊叹”；追欧洲惊悚电影，“迅速沉浸在故事中”；还爱在日本胜沼酒庄品酒，那里出产极品白酒。他对自己科学的最爱是“随心所欲创造任何纳米级网络结构”。这些爱好不仅是休憩，更是灵感来源——正如庄子所言，无用之用，往往孕育奇迹。

如今，北川进身兼京都大学三项职位，其中一项名誉教授无强制退休年龄。他无意退休，继续领导iCeMS，探索MOFs的第三代应用。从京都少年到诺贝尔得主，他的故事告诉我们：科学不仅是公式，更是挑战自我、拥抱未知的冒险。正如他在发布会上的寄语，挑战，将点亮更多年轻的心灵。
