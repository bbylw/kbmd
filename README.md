亲爱的乌克兰人!

在第聂伯罗，俄罗斯无人机袭击后，救援工作仍在进行中——这是一次蓄意的袭击，目标是市中心。不幸的是，有人受伤。

总的来说，一整天，包括哈尔科夫、第聂伯罗地区、苏梅地区和切尔尼希夫地区在内的东部地区，都处于近乎持续的空中威胁之下。哈尔科夫遭受了袭击。切尔尼希夫地区的一个变电站被击中，修复工作正在进行中。

在苏梅地区，夜间俄罗斯人杀害了一整个家庭，就在他们的家中：一名妻子、一名丈夫和两个年幼的孩子，他们分别是四岁和七岁。这位女士当时怀有身孕。

这是一次俄罗斯无人机袭击。

我对他们的家人和亲人表示哀悼。我们一定会对俄罗斯的袭击作出回应。

今天也有军事报告，特别是关于我们的纵深打击。我们将继续，完全公正地，摧毁俄罗斯的后勤，俄罗斯的燃料基础设施——所有维持俄罗斯战争的一切。

全世界都知道，战争没有结束的责任只在俄罗斯。我们正在收到来自美国和我们其他合作伙伴的正确信号——这些信号表明，需要力量来迫使俄罗斯进行真正的外交和谈判，进行和平会议——这些形式才能真正有所帮助，而俄罗斯却一直在躲避。

没有强大的压力，俄罗斯人是不会停止的；相反，他们试图升级侵略。

现在是第七天了——顺便说一句，这是以前从未发生过的事情——扎波罗热核电站处于紧急状态。情况危急。由于俄罗斯的炮击，核电站已被切断电源，与电网断开，目前正由柴油发电机供电。

这是非同寻常的。发电机和核电站并非为此设计，也从未如此长时间地以这种模式运行。而且我们已经有信息表明，其中一个发电机已经出现故障。正是俄罗斯的炮击阻止了电线对核电站的修复以及基本安全的恢复。

这对所有人都是一个威胁。世界上没有任何恐怖分子敢于对核电站做出俄罗斯现在正在做的事情。世界不保持沉默是正确的。

我与军方和能源部举行了会议。我已经指示政府——乌克兰能源部，与外交部一起——要让全世界充分关注这一局势。

我还与国家安全和国防委员会秘书以及总统制裁政策专员举行了制裁会议。任务很明确：制裁必须给俄罗斯带来更大的痛苦，更切实的后果。

在十月，制裁同步工作必须加快——特别是要协调我们在伙伴司法管辖区内的制裁。我们期待欧盟通过第19轮制裁方案，这也将是与美国人进行有力讨论的一个要点——以确保他们的压力不会落后于欧洲和其他七国集团成员的压力。

我还指示，为延长即将到期的制裁做准备。

到目前为止，九月份我们已经推出了五个新的制裁方案。我们的合作伙伴也做出了非常有帮助的决定。在许多情况下，欧洲的制裁、加拿大的制裁以及美国的制裁都是基于我们乌克兰的提议。我们对此非常感谢我们的合作伙伴。

今天我与联合国秘书长古特雷斯通了电话。我感谢他的支持，并概述了我们为接回被俄罗斯扣押的乌克兰囚犯、我们的成年人以及被关押在俄罗斯的乌克兰儿童所做的努力。

我们正在为联大准备一项关于被俄罗斯绑架儿童的决议。我们正在与各国合作，以确保有效的投票，最重要的是——我们的国民回归，我们的孩子回归。

还有一件事。今天，乌克兰情报部门与外交部一起，从加沙又疏散了57人——其中48人是乌克兰公民，包括16名儿童。

总共近两年来，通过我们的努力，已有422人从加沙撤离——不仅是乌克兰公民，还有邻国的公民。

多亏了美国的倡议，现在有一项协议，致力于解决整个加沙的局势——这不仅对该地区很重要，对全世界许多其他地方也意义重大。

特朗普总统的提议很重要——我们今天也与联合国秘书长讨论了这些提议。这是一个达成和解的真正机会。我们已准备好为促成美国的提议成功而做出贡献。

我感谢所有帮助保护生命的人，所有恢复国际法效力以及尊重国家、尊重人的生命和停止战争的国际准则的人。这是最重要的。

荣耀归于乌克兰！
