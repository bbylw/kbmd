我祝愿各位乌克兰同胞健康！

今天一整天，我们都在进行恢复工作，以应对俄罗斯的袭击——在扎波罗热、基辅、基辅地区以及我国其他地区。大多数打击目标是普通的居民楼和民用设施。超过80人受伤。遗憾的是，有4人丧生，其中包括一名儿童。我向他们的家人和亲人致以哀悼。

在基辅，受损设施中包括斯特拉热斯科心脏病研究所——我国以及欧洲地区领先的医疗机构之一。该处有两人丧生，一栋建筑遭受严重损坏。必须尽快进行修复，政府应提供援助。仅在基辅，就有32处地点受损。

俄罗斯对扎波罗热的打击极其残暴。幸运的是，没有人员死亡，但仅在扎波罗热地区，昨晚就有近40人受伤。他们所有人都在接受必要的援助。

我想感谢所有参与救援行动的人——我们国家紧急事务服务局的工作人员、紧急医疗服务人员、医生、护士、警察以及市政服务部门。

俄罗斯还袭击了能源设施。不幸的是，这已成为俄罗斯的传统战术——俄罗斯再次试图今年让乌克兰陷入停电。

事实上，在这些周里，当联合国大会举行期间，俄罗斯几乎每天、每小时都在打击乌克兰。极其卑鄙的打击。这就是俄罗斯的本质——他们所做的事，而不是他们所说的话或向世界发出的信号。

他们拒绝了所有真正的和平提议，并应受到真正严厉的压力。本周，我们期待欧洲就此类压力做出决定——至少应出台第19个制裁方案。特别重要的是，制裁应严厉打击俄罗斯的能源贸易及其整个油轮舰队的相关基础设施。而且，情报显示，俄罗斯人正在利用油轮发射和操作针对欧洲国家的无人机。这进一步证明，为什么波罗的海和其他海域必须对俄罗斯油轮关闭——至少是对影子舰队。

我们也期待美国采取强有力的步骤。我与特朗普总统讨论了什么能真正推动俄罗斯改变立场并停止战争。世界有能真正发挥和平作用的工具。

今天，我与芬兰总统通话，并感谢他的支持。我们协调了与伙伴的联系，特别是与美国。重要的是，我们要稳步推进最终确定安全保障——那些既能帮助结束这场战争，又能威慑俄罗斯进一步侵略的措施。

我还与挪威首相通话。我们讨论了今天的俄罗斯袭击——超过500架无人机，其中约350架是“沙赫德”无人机，以及48枚导弹，包括两枚空射弹道导弹。这就是为什么防空始终是乌克兰的首要优先事项。我感谢提供帮助的伙伴。挪威就是其中之一，是我们最坚定的伙伴之一。谢谢你，乔纳斯！

今天，我还与北约秘书长马克·吕特通话。我们讨论了其他事项，其中包括我们的PURL（乌克兰优先需求清单）倡议。该倡议正在发挥作用，提供非常积极的援助——它正是允许我们用北约伙伴的资金购买美国武器的机制。这些包括爱国者导弹和HIMARS导弹。

已有六个国家参与PURL——荷兰、丹麦、瑞典、挪威、德国和加拿大。另外六个国家已表示愿意加入——立陶宛、拉脱维亚、爱沙尼亚、卢森堡、比利时和冰岛。我感谢他们所有人。这才是真正有帮助的。

本周，我们将与欧洲朋友合作，尽快落实防空协议。俄罗斯肯定将为其所做的一切负责。

感谢世界各地所有帮助乌克兰的人——我们的民众、我们的国家以及我们的独立。

荣耀归于乌克兰！
